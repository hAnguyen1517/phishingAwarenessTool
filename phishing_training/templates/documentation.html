Phishing Training Platform Documentation

Last updated: March 12, 2025

Overview

The Phishing Training Platform is a web-based application designed to educate and train users in identifying and preventing phishing attacks. This documentation provides detailed information for administrators, regular users, and special accounts (e.g., superusers) to effectively use and manage the system.

System Architecture

The platform is built using a client-server architecture with the following components:

  • Frontend: HTML, CSS, and JavaScript with a responsive design.
  • Backend: Django framework, handling user authentication and data processing.
  • Database: MongoDB for storing user data, training progress, and reports.
  • API: RESTful API for communication between frontend and backend.

Deployment: Hosted on AWS with load balancing and SSL encryption for security.

User Roles and Access Levels

1. Administrators

Administrators have extensive rights to manage the platform, including:

  • Create, edit, and delete user accounts.
  • Generate system-wide reports and analytics.
  • Configure training modules and simulations.

Access Instructions: Log in with admin credentials and navigate to the "Reports" or "Settings" sections.

2. Regular Users

Regular users have standard access to:

  • Access training modules and quizzes.
  • View personal progress and reports.
  • Update their profile information.

Access Instructions: Sign in with user credentials and explore the "Training" and "Quiz" sections.

3. Special Accounts (Superusers)

Superusers or system accounts have advanced privileges, including:

  • Oversee all user activities and system logs.
  • Perform system maintenance and updates.
  • Reset user accounts or troubleshoot issues.

Access Instructions: Contact support at support@phishingtool.com for superuser access.

Key Features and Usage

  • Interactive Simulations: Users can engage in simulated phishing emails to test their skills.
  • Real-Time Reporting: Admins and users can access dashboards with performance metrics.
  • Comprehensive Training: A library of courses is available for all users.

Technical Requirements

  • Browser: Latest versions of Chrome, Firefox, or Edge.
  • Internet: Stable connection with at least 5 Mbps speed.
  • Device: Desktop, tablet, or mobile with a minimum screen resolution of 1024x768.

Troubleshooting and Support

For issues, refer to the Help Center or contact support at support@phishingtool.com.
